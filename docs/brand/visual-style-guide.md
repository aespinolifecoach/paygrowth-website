# PayGrowth Brand Identity & Visual Style Guide

**Brand Name:** PayGrowth

## 4. Logo System

### Primary Logo Versions

#### Full Logo (Horizontal)
- Icon + wordmark combined horizontally
- Minimum width: 180px
- Clear space: 1x icon height on all sides

#### Icon-Only Mark
- Standalone symbol for favicons, app icons, social avatars
- Minimum size: 32px × 32px
- Must maintain legibility at 16px

#### Stacked Logo
- Icon above wordmark for square placements
- Used in: app splash screens, business cards, square media

### Color Variations

| Variation | Background | Usage |
|-----------|------------|-------|
| **Primary** | White/Navy logo on white/light backgrounds | Standard usage |
| **Reversed** | White logo on Midnight Navy backgrounds | Dark sections |
| **Accent** | Teal/Glow effects on dark mode | Special emphasis |
| **Monochrome** | Single-color logo for restricted palettes | Single-color print |

### Logo Misuse (Prohibited)
- Do not stretch or distort the logo
- Do not change logo colors outside approved variations
- Do not add shadows, effects, or filters to logo
- Do not place logo on busy backgrounds without proper contrast
- Do not recreate or modify the logo symbol
- Minimum clear space must be maintained

## 5. Iconography

### Icon Style
- **Style:** Outlined, rounded line icons
- **Stroke Width:** 1.5px - 2px consistent
- **Corner Radius:** 2px for sharp details, 4px for soft elements
- **Grid:** 24px × 24px base grid for all icons

### Core Icon Set

| Category | Icons |
|----------|-------|
| **Payment Methods** | Credit card, Debit card, Bank transfer, Wallet |
| **Commerce** | POS terminal, Receipt, Invoice, Shopping cart |
| **Growth** | Chart up, Trend arrow, Target, Star rating |
| **Security** | Shield, Lock, Verified, Checkmark circle |
| **Communication** | Phone, Email, Chat, Bell |

### Custom Icon Treatment
All icons should use the Electric Teal (`#00D4AA`) accent for interactive states.

## 6. Imagery & Photography

### Photography Style
- **Subject:** Business owners, merchants, clean workspaces, modern retail
- **Tone:** Professional yet approachable, diverse representation
- **Lighting:** Natural, bright, clean — avoid heavy shadows
- **Composition:** Modern crops, slight depth of field acceptable

### Image Treatments
- Subtle gradient overlays using brand gradient colors
- Light vignettes acceptable for depth
- Consistent color grading with slight cool bias

### Stock Photography Sources
- Unsplash (preferred)
- Pexels
- Custom photography for key campaign assets

## 8. Website & App UI

### Web Color Usage

| Element | Color | Hex |
|---------|-------|-----|
| **Primary Background** | White | `#FFFFFF` |
| **Secondary Background** | Off-White | `#F8FAFC` |
| **Primary Text** | Charcoal | `#1E293B` |
| **Secondary Text** | Dark Gray | `#475569` |
| **Links** | Bright Cyan | `#00B4D8` |
| **Link Hover** | Electric Teal | `#00D4AA` |
| **Primary Buttons** | Electric Teal | `#00D4AA` |
| **Button Text** | White | `#FFFFFF` |
| **Card Borders** | Light Gray | `#E2E8F0` |
| **Section Dividers** | Light Gray | `#E2E8F0` |
| **Dark Sections** | Midnight Navy | `#0A1628` |
| **Dark Section Text** | White | `#FFFFFF` |

### Component Standards

#### Buttons
- **Primary:** Electric Teal background, White text, 8px border-radius
- **Secondary:** Navy background, White text, 8px border-radius
- **Outline:** Transparent with Navy border, Navy text
- **Hover:** 10% darker, subtle lift shadow
- **Height:** 44px minimum (touch-friendly)

#### Cards
- **Background:** White or Off-White
- **Border:** 1px Light Gray or none (elevation shadow)
- **Border Radius:** 12px
- **Shadow:** `0 4px 6px -1px rgba(10, 22, 40, 0.1)`
- **Padding:** 24px internal

#### Form Elements
- **Input Height:** 44px
- **Border:** 1px Medium Gray
- **Focus Border:** 2px Electric Teal
- **Border Radius:** 8px
- **Label:** 14px Semi-bold, Dark Gray

## 9. Brand Voice & Tone

### Voice Attributes
- **Professional:** Expert in payments, reliable partner
- **Clear:** Simple language, no jargon
- **Approachable:** Friendly, supportive, encouraging
- **Forward-looking:** Growth mindset, innovation

### Messaging Principles
1. **Lead with benefit:** "Grow your business" not "We process payments"
2. **Be specific:** Concrete numbers and outcomes
3. **Show empathy:** Understand merchant challenges
4. **Build trust:** Emphasize security and reliability

### Example Copy

| Context | Tone |
|---------|------|
| **CTAs** | "Start Growing Today" / "Get Started" |
| **Features** | "Fast, Secure Payment Processing" |
| **Trust** | "Bank-Level Security" / "PCI Compliant" |
| **Support** | "We're Here to Help" / "24/7 Support" |
